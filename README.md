# Image downloader

## Caratteristiche:

1. Scarica WebP, PNG, JPEG, JPG e GIF
2. Gestisce URL relativi e assoluti
3. Evita duplicati e sovrascritture
4. Headers browser per evitare blocchi
5. Pausa configurabile tra download
6. Gestione errori robusta

## Come usarlo:

1. Salva come image_downloader.py
2. Esegui: python image_downloader.py
3. Inserisci l'URL del sito
4. Scegli la cartella di destinazione

## Dipendenze automatiche:
Lo script installa da solo requests e beautifulsoup4 se mancanti.
